# Security Policy

## Supported Versions

The `main` branch receives security fixes for the current release-candidate package.

## Reporting a Vulnerability

Please report suspected vulnerabilities through GitHub Security Advisories or by opening a private issue with reproduction details.

Do not include unreleased launch plans, private repository evidence, API tokens, customer data, or social account credentials in reports. Redact sensitive source material before attaching examples.

## Scope

`postmaker-skill` is a local-only draft generator. It reads supplied evidence files and prints draft content. It should not publish posts, send messages, call social APIs, or write to external accounts.
